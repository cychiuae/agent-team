# Docs

Single source of truth for this project. Code is a derivative artifact — when behavior changes, the relevant doc here is updated **first**.

## Subdirectories

- [plans/](plans/index.md) — per-feature planning artifacts orchestrated by `/ship`.

## Conventions

- Every directory under `docs/` has its own `index.md` table of contents. When you add, rename, or remove a file, update the matching `index.md` in the same change.
- See the [`requirements-doc-format`](../.claude/skills/requirements-doc-format/SKILL.md) skill for the full directory contract, file format, requirement-ID rules, and what does **not** belong in `docs/product-spec/`.
