# Plans

Per-feature planning artifacts orchestrated by `/ship`. Each feature gets its own subdirectory (`docs/plans/<slug>/`) following the layout in [`_template/`](_template/).

## Subdirectories

- [_template/](_template/) — canonical artifact set copied for each new feature: exploration, change summary, task files, status, and final verification.

When a new feature plan is started, add it here with a one-line description.
